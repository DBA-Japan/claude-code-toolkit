# references/

Anonymized reference files for web development and Claude Code tooling. Based on field research (36-agent survey, 20-company design review, 2026-03).

## Files

### web-libraries.md
The complete library catalog: 50+ entries with CDN URLs, star counts, and use cases. Covers animation (GSAP, Motion), scrolling (Lenis), particles, cursors, carousels, loaders, and more. Also includes:
- Design auto-selection guide (Step 1/2/3: industry → personality → target)
- Performance rules (10 mandatory rules)
- CSS-only new features table (no JS needed)
- AI video generation (Higgsfield AI) with LP embed code
- Free design resources table (Grainient, Fontshare, remove.bg, etc.)

### design-resources.md
X/Twitter accounts to follow for CSS and web design inspiration, with code snippets from top creators. Includes:
- CSS animation accounts (@jh3yy, @ChallengesCss, @argyleink, @anatudor)
- Inspiration feeds (@codrops, @greensock, @awwwards)
- Bento Grid / layout specialists
- jhey's CSS Tips code collection (glow borders, gradient rotation, scroll-driven cards)
- Free design resource table
- Reference CodePen links

### design-patterns.md
Standalone quick-reference for 8 proven design patterns extracted from real Japanese IT company websites. Each pattern includes the visual description, reproduction code, and reference site. Patterns cover: clean motion, photo shape variation, dual-background parallax, cursor follower, zoom slide, background text decoration, animated symbol + texture, and dot background color transition. Also includes a combination guide by company type.

### cc-ecosystem.md
Claude Code ecosystem tools beyond the built-in CLI. Covers:
- Recommended community resources (everything-claude-code, claude-peers-mcp)
- Skill discovery platforms: SkillsMP, aitmpl.com, SkillHub, agentskills.io — with a comparison table
- Task Master AI MCP (PRD → tasks.json automation, 25k+ Stars)
- Remotion (programmatic video generation, official CC skill)
- NotebookLM MCP (Google NotebookLM via Claude Code, no API key needed)
- Quick reference install command table
