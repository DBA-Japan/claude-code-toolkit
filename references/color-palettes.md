# カラーパレット 2026 トレンド

2025-2026 年の Web 色トレンドと、業種別の非定番パレット提案。

---

## トレンド概要

- **Pantone 2026**: Cloud Dancer (#F5F3EF) — 温かみのあるニュートラル
- **ネオン**: マイクロアクセントとして復活
- **ブルーグリーン**: 自然×テック融合の 2026 年注目色
- **テラコッタ / ラスト**: 大地的温かみ
- **シネマティックグラデーション**: ソフトグロー、メッシュ、ambient lighting

## ダークモード推奨色

| 用途 | 色 | 理由 |
|------|-----|------|
| 背景 | `#0F0F0F` | つや消し黒。`#000` は避ける（眩しすぎる） |
| カード | `#1A1A1A` | 微妙な浮遊感 |
| テキスト | `#E8E8E8` | オフホワイト（`#FFF` は避ける） |
| ボーダー | `rgba(255,255,255,0.08)` | ほぼ見えないが構造を作る |

ネオンアクセントがダーク背景で最も映える。

---

## 白ベースのアクセント使い方

1. **2-3 色に絞る**（多色は散漫）
2. **CTA に最も鮮やかな色**
3. **セクション分けに中間色**
4. 純白 `#FFF` ではなく Cloud Dancer 系 `#F5F3EF` を使う（温かみ）
