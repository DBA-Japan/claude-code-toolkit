# /admin-docs — 行政書類・申請書の下書き作成

旅館業に関する行政書類、許認可申請、届出書類の下書きを作成します。

## 使い方

```
/admin-docs [書類の種類]
```

種類の例:
- `旅館業許可` — 旅館業許可申請書
- `消防届出` — 消防法に基づく届出
- `補助金` — 補助金・助成金申請書
- `事業計画` — 事業計画書（融資・補助金用）
- `届出変更` — 変更届出書

## 動作

### Step 1: 書類の特定
- 必要な書類の種類を確認
- 提出先（市役所、保健所、消防署、観光庁等）
- 期限の確認

### Step 2: 必要情報の収集
質問形式で必要な情報を聞き取り:
- 施設名、所在地、代表者名
- 施設の概要（構造、面積、定員）
- 営業形態（簡易宿所、旅館、住宅宿泊事業）
- 関連する設備情報

### Step 3: 下書き生成

#### 事業計画書（融資・補助金用）
```markdown
# 事業計画書

## 1. 事業概要
- 事業名:
- 所在地:
- 事業形態:
- 代表者:

## 2. 事業の目的
（地域活性化、空き家活用、観光促進等の社会的意義）

## 3. 市場分析
- 対象地域の観光客数推移
- 競合施設の状況
- ターゲット顧客層

## 4. 収支計画
| 項目 | 年1 | 年2 | 年3 |
|------|-----|-----|-----|
| 売上 | | | |
| 経費 | | | |
| 利益 | | | |

## 5. 資金計画
- 自己資金:
- 借入金:
- 補助金:

## 6. 実施スケジュール
```

#### 補助金申請書
- 地域の補助金制度に合わせた書式
- 申請理由、事業効果、地域貢献を明記

### Step 4: 出力
- **Markdown**（編集用）
- **PDF**（提出用）

## 注意事項
- **法的責任**: この出力はあくまで「下書き」です。最終的な内容は行政書士や専門家に確認してください
- **最新法令**: 旅館業法、住宅宿泊事業法（民泊新法）は改正が頻繁です。最新の法令を確認してください
- **地域ごとの差異**: 自治体によって必要書類や基準が異なります
