# /operation-manual — 運営マニュアル・チェックリスト作成

宿泊施設の運営に必要なマニュアル、チェックリスト、手順書を作成します。

## 使い方

```
/operation-manual [マニュアルの種類]
```

種類の例:
- `チェックイン` — チェックイン手順書
- `清掃` — 清掃チェックリスト
- `緊急対応` — 緊急時対応マニュアル
- `設備` — 設備トラブルシューティング
- `新人` — 新人スタッフ研修マニュアル
- `季節` — 季節別運営ガイド

## 動作

### Step 1: 要件確認
- 宿の名前と特徴
- 対象スタッフ（オーナー / 清掃スタッフ / 外部委託）
- 既存のルールやこだわり

### Step 2: マニュアル生成

#### チェックイン手順書
```
□ ゲスト到着30分前: エアコンON、照明点灯
□ 鍵の受け渡し方法の確認（キーボックス / 対面）
□ 館内案内（5分以内）
  - WiFiパスワード
  - ゴミ分別ルール
  - 緊急連絡先
  - チェックアウト時間
□ 近隣飲食店マップを渡す
□ 「何かあればいつでもLINEください」と伝える
```

#### 清掃チェックリスト
```
【リビング】
□ 掃除機
□ テーブル・棚の拭き掃除
□ ゴミ回収
□ リモコン・スイッチ類の消毒

【キッチン】
□ シンク洗浄
□ コンロ掃除
□ 冷蔵庫内チェック（忘れ物）
□ 食器・調理器具の確認（不足補充）
□ ゴミ袋セット

【浴室・トイレ】
□ 浴槽・シャワー洗浄
□ トイレ清掃・消毒
□ タオル交換（バス×2、フェイス×2、ハンド×2 / 人数分）
□ アメニティ補充

【寝室】
□ シーツ・枕カバー交換
□ 布団の状態確認
□ エアコンフィルター確認（月1回）

【最終チェック】
□ 忘れ物確認（全部屋）
□ 窓の施錠
□ 写真撮影（清掃完了の記録）
```

#### 緊急対応マニュアル
- 停電、断水、設備故障の連絡先と手順
- 台風・地震時の対応
- ゲストの体調不良時の対応
- 近隣トラブル

### Step 3: 出力形式
- **Markdown**（Claude Codeで管理）
- **PDF**（印刷用。現場に貼る用）
- **チェックリスト**（毎回使うもの）

## カスタマイズ
宿ごとに設備や運営方針が異なるため、ベーステンプレートを生成後、宿固有の情報を追加してください。
