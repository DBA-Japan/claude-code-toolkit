# /sns-post — SNS 投稿・マーケティング文面生成

Instagram / X(Twitter) / Facebook / LINE公式アカウント向けの投稿文を、宿の魅力を最大限に伝える形で生成します。

## 使い方

```
/sns-post [プラットフォーム] [テーマ]
```

例:
```
/sns-post Instagram 桜の季節
/sns-post LINE 新プラン告知
/sns-post X 週末の空き状況
```

## 動作

### Step 1: 投稿設計
- **プラットフォーム**: Instagram / X / Facebook / LINE
- **目的**: ブランディング / 予約促進 / 情報提供 / エンゲージメント
- **テーマ**: 季節、イベント、宿の日常、ゲストの声、地域情報

### Step 2: プラットフォーム別生成

#### Instagram
- **キャプション**: 3-5行 + 改行 + ハッシュタグ
- **ハッシュタグ**: 地域名 + 宿泊スタイル + 季節 + 体験（15-20個）
- **トーン**: 温かく、写真の世界観に合わせる
- **CTA**: 「プロフィールのリンクからご予約いただけます」

```
築130年の古民家で迎える、佐渡島の春。

障子越しに差し込む朝日、
庭に咲き始めた桜、
囲炉裏端で飲む朝のコーヒー。

ここにしかない「暮らし」を、泊まりに来ませんか。

#佐渡島 #古民家ステイ #一棟貸し #暮らしに泊まる
#sado #japantravel #traditionaljapan ...
```

#### X (Twitter)
- **文字数**: 140文字以内
- **画像**: 1-2枚推奨
- **トーン**: カジュアルだが品がある

#### LINE公式アカウント
- **構成**: 挨拶 + 本文 + CTA
- **特典**: LINE限定割引やお知らせ
- **配信頻度**: 月2-4回（多すぎるとブロックされる）

#### Facebook
- **長文OK**: ストーリー仕立て
- **写真アルバム**: 複数枚
- **地域コミュニティへの投稿も検討**

### Step 3: コンテンツカレンダー提案
月間の投稿スケジュール案を生成:

```
| 週 | プラットフォーム | テーマ | 目的 |
|----|----------------|--------|------|
| 第1週 | Instagram | 季節の風景 | ブランディング |
| 第2週 | LINE | 空き状況 | 予約促進 |
| 第3週 | Instagram | ゲストの声 | 信頼構築 |
| 第4週 | X | 地域イベント | エンゲージメント |
```

## 写真の撮り方ガイド
- **朝の光**: 7-9時の自然光が最強
- **生活感**: 食卓、囲炉裏、布団 → 「暮らし」を感じる
- **人を入れる**: 後ろ姿、手元 → 温かみ
- **引きと寄り**: 外観（引き）→ ディテール（寄り）のセット
- **縦構図**: Instagram はスクエアか縦長
