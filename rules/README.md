# rules/

汎用的な「絶対ルール」集。`install.sh` が `~/.claude/rules/` にコピーします。

各ルールは独立した `.md` で、`~/CLAUDE.md` から相互参照される設計です。

## 含まれるもの

| Rule | 用途 |
|---|---|
| `safety.md` | 不可逆操作は事前許可・分析と修正の境界・ブラックボックス禁止 |
| `factcheck.md` | 統計・数字は提示前に原典確認、未検証マーク |
| `no-dashes.md` | ダッシュ・ハイフン使用禁止（AI 文体回避） |
| `no-truncation.md` | 大規模生成時の途中省略禁止 |
| `edit-policy.md` | 手動編集尊重・パッチ方式・「一から作り直す」は最終手段 |
| `security-discipline.md` | シークレットアクセス禁止・MCP 書込前確認・プロンプトインジェクション対策 |

## カスタマイズ

各ファイルは「文書として」読み込まれて Claude が遵守します。プロジェクトの状況に合わせて中身を書き換えて構いません。
