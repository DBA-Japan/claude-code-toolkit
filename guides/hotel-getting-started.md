# ホテル・宿泊施設経営者のための Claude Code 入門ガイド

Claude Code を宿泊施設の運営に活用するための最初の一歩。プログラミング経験は不要です。

---

## Claude Code って何？

ターミナル（黒い画面）で動く AI アシスタント。話しかけるだけで、文書作成・データ分析・メール返信・レポート生成などを行います。ChatGPT との違いは、**ファイルを直接読み書きできる**こと。

例: 「先月の予約データを分析して PDF レポートを作って」と言うだけで、データを読んで分析してレポートを生成します。

---

## セットアップ（5 分）

### Step 1: Claude Code を起動
ターミナルを開いて `claude` と入力。

### Step 2: 最初に試すこと
以下をコピペして実行してみてください:

```
今月の業務で一番時間がかかっていることは何ですか？と聞いてくるので、答えてください。その上で効率化の提案をしてください。
```

### Step 3: スキルを使ってみる
このツールキットに入っているスキル（コマンド）を試す:

```
/guest-reply 明日チェックインですが、駐車場はありますか？
```

---

## 宿泊施設で使えるスキル一覧

| コマンド | できること | こんな時に使う |
|---------|-----------|--------------|
| `/guest-reply` | ゲストメッセージの返信ドラフト生成 | Airbnb/Booking.com の問い合わせ対応 |
| `/ota-listing` | OTA 物件説明文の最適化 | Airbnb の説明文を改善したい時 |
| `/review-reply` | レビュー返信の生成 | ゲストのレビューに返信する時 |
| `/operation-manual` | 運営マニュアル・チェックリスト作成 | 清掃手順書、緊急対応マニュアル |
| `/business-report` | 月次レポート・競合分析 | 売上データの分析、事業計画 |
| `/sns-post` | SNS 投稿文の生成 | Instagram、LINE の投稿作成 |
| `/admin-docs` | 行政書類の下書き | 補助金申請、届出書類 |
| `/chief-of-staff` | メッセージの仕分け・返信提案 | 大量のメール/LINE を処理 |
| `/save-session` | 作業状態を保存 | 途中で中断する時 |
| `/resume-session` | 前回の作業を復元 | 昨日の続きをやる時 |
| `/aside` | 作業中に別の小タスクを処理 | 予約対応中に急ぎの質問が来た時 |

## よくある使い方

### 1. ゲスト対応を効率化
```
/guest-reply

「Hello, we are arriving tomorrow at 3pm.
Is there parking available? Also, can we use the kitchen?」
```
→ 英語で丁寧な返信ドラフトが生成される

### 2. レビューに返信
```
/review-reply

「とても素敵な古民家でした。特に囲炉裏が印象的。
ただWiFiが少し遅かったのが残念。★★★★☆」
```
→ 感謝 + WiFi 改善の対応を含む返信が生成される

### 3. 月末のレポート作成
```
/business-report 月次

今月の予約数: 18件
Airbnb: 12件（売上 ¥480,000）
Booking.com: 4件（売上 ¥160,000）
自社サイト: 2件（売上 ¥80,000）
キャンセル: 2件
```
→ 分析付きの月次レポートが自動生成される

### 4. 清掃チェックリストを作る
```
/operation-manual 清掃

東夢（築130年古民家、4LDK、定員10名）
特に注意: 囲炉裏の灰、古い障子の扱い、BBQ機材の洗浄
```
→ 宿に合わせたチェックリストが生成される

## コンテキスト管理（大事）

Claude Code は「机の広さ」に限りがあります。長く使っていると処理が遅くなる場合があります。

**対処法**:
- `/compact` と打つ → 過去の会話を圧縮して机を片付ける
- 新しいセッションを開始する → `claude` で新規スタート

---

## 困った時

| 症状 | 対処 |
|------|------|
| 返答が遅い | `/compact` で圧縮 |
| 前回の作業を忘れた | `/resume-session` で復元 |
| 何を聞けばいいか分からない | 「今の業務で困っていることは○○です。助けて」と自然に話す |
| エラーが出た | エラー文をそのままペーストして「これ何？」と聞く |
